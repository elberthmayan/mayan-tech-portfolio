# 🚀 Guia de Hospedagem - Mayan Tech

Parabéns pelo projeto! O site está pronto para o mundo. Aqui estão os passos para colocar ele online de forma profissional e gratuita.

## 1. Hospedagem Recomendada: Vercel (Grátis)
A Vercel é a melhor opção para sites estáticos como este.

1.  Crie uma conta em [vercel.com](https://vercel.com).
2.  Instale a Vercel CLI no seu computador ou conecte seu repositório do GitHub.
3.  Se usar a CLI:
    *   Abra o terminal na pasta do projeto.
    *   Digite `vercel` e siga as instruções (pode dar "Enter" em tudo).
    *   O site estará online em segundos!

## 2. Ajuste Final de SEO
Após hospedar, você receberá um link (ex: `meu-site.vercel.app`). 
Para que a pré-visualização no WhatsApp fique perfeita, abra o `index.html` e altere a linha:
` <meta property="og:image" content="./img/Doceria.webp">` 
Para o link completo:
` <meta property="og:image" content="https://meu-site.vercel.app/img/Doceria.webp">`
